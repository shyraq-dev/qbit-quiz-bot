## ✅ НЕ ТҮЗЕТІЛДІ:

### 1️⃣ ХАТ ЖІБЕРУ — HTML + VARIABLES

**HTML пішімдеу қолдауы:**
```html
<b>қалың</b> — қалың мәтін
<i>көлбеу</i> — көлбеу
<u>астын сызу</u> — астын сызылған
<s>үстін сызу</s> — үстін сызылған
<code>код</code> — код блогы
<pre>блок</pre> — код блогы (көп жолды)
<a href="url">сілтеме</a> — гиперсілтеме
```

**Айнымалылар (персоналдау):**
```
{ID} — 123456789
{NAME} — Асет
{SURNAME} — Нұрланұлы
{NAMESURNAME} — Асет Нұрланұлы
{USERNAME} — @aset_nrln
{LANG} — kk
{DATE} — 2026-02-23
{TIME} — 15:30
{WEEKDAY} — Жұма
{MENTION} — <сілтеме профильге>
```

**Мысал:**
```html
<b>Сәлем, {NAME}!</b>

Сіздің балыңыз: <u>98/100</u> 🎉

Келесі тест: <i>{WEEKDAY}, {TIME}</i>

<a href="https://t.me/qbitquiz">Ботқа өту</a>
```

**Өзгерістер:**
- `parse_mode="HTML"` қолданылады (Markdown емес!)
- `replace_variables()` функциясы қосылды
- Әр қолданушыға жеке-жеке хабар (персоналдау)
- `get_user_by_id()` функциясы database.py-ға қосылды
