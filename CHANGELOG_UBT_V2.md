## 🗄️ DATABASE

### Жаңа кесте:

```sql
CREATE TABLE ubt_questions (
    id              INTEGER PRIMARY KEY,
    subject_id      INTEGER,
    question_type   TEXT,  -- simple/context/matching/multiple
    question_text   TEXT,
    context_text    TEXT,

    -- Simple/Context/Multiple
    option_a, option_b, option_c, option_d TEXT,
    correct         TEXT,
    correct_multiple TEXT,

    -- Matching
    matching_left   TEXT,  -- JSON
    matching_right  TEXT,  -- JSON
    matching_answer TEXT,  -- JSON

    image_url       TEXT,
    difficulty      TEXT,
    created_at      TEXT
);
```

### Өзгертілген:
- `ubt_subjects`: questions_count түзетілді (10/10/20/40)
- `ubt_sessions`: score бағандары қосылды
