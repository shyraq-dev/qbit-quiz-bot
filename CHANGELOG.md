# QBit Quiz Bot — Өзгерістер журналы

## v1.7 — Хат жіберу (Broadcast) (2026-02-23)

### ✨ Жаңа мүмкіндіктер

#### 📩 Хат жіберу (Broadcast)

**Админ функциялары:**

**1. Хабарлама түрлері:**
- 📝 **Мәтін** — қарапайым текст
- 🖼 **Сурет** — photo + caption
- 🎥 **Видео** — video + caption
- 📄 **Файл** — document + caption

**2. Алушылар:**
- 👥 **Барлығына** — барлық тіркелген қолданушылар
- ✅ **Белсенділерге** — соңғы 7 күнде белсенді болғандар

**3. Процесс:**
1. Админ панелі → **📩 Хат жіберу**
2. Түрін таңда (мәтін/сурет/видео/файл)
3. Алушыларды таңда (барлығы/белсенділер)
4. Мәтінді жаз
5. Media URL (егер сурет/видео/файл болса)
6. Алдын ала қарау
7. `/send` — жіберу

**4. Нәтиже:**
- Real-time статус (жіберілуде...)
- Жалпы/Жіберілді/Қате саны
- Broadcast ID тарих үшін

**5. Жіберу тарихы:**
- Админ панелі → **📊 Жіберу тарихы**
- Соңғы 5 жіберу
- ID, түрі, мәтін, статистика, уақыт

#### 📝 Қосымша материалдар

1. **🖼 Сурет** (image_url):
   - URL арқылы сурет қосу
   - Сурет сұрақпен бірге көрсетіледі
   - Photo message түрінде жіберіледі

2. **💬 Мәнмәтін** (context_text):
   - Қосымша түсініктеме немесе контекст

- Админ сұрақ қосқанда екі қосымша қадам:
  - Мәнмәтін қосу (міндетті емес, /skip)
  - Сурет URL қосу (міндетті емес, /skip)
- Quiz кезінде:
  - Сурет болса: photo + caption ретінде
  - Мәнмәтін болса: сұрақтан бұрын "💬 Мәнмәтін:" деп көрсетіледі

## v1.4 — ҰБТ режимі (2026-02-23)

### ✨ Жаңа мүмкіндіктер

#### 📚 ҰБТ режимі

**🎓 Құрылымы:**
- **Міндетті пәндер** (3 пән × 25 сұрақ = 75):
  - Оқу сауаттылығы (25 сұрақ)
  - Математикалық сауаттылық (25 сұрақ)
  - Қазақстан тарихы (25 сұрақ)

- **Таңдау пәндері** (2 пән × 20 сұрақ = 40):
  - Қолданушы 2 пән таңдайды
  - Әр пәннен 20 сұрақ

**⏱ Режимдер:**
- 📝 **Толық тест:** 180 минут (3 сағат) — 115 сұрақ
- ⚡ **Қысқартылған:** 60 минут — 115 сұрақ

**🎯 Балл жүйесі:**
- Жалпы сұрақтар: 115
- Максимум балл: 140
- Коэффициент: 1 дұрыс ≈ 1.217 балл
- Бөлінісі:
  - Міндетті пәндер баллы
  - Таңдау пән 1 баллы
  - Таңдау пән 2 баллы
  - Жалпы балл

**📊 Нәтиже:**
- Пән бойынша дәлдік
- Жалпы дәлдік
- Балл (0-140)
- Бейдж: 🏆 Өте жақсы / ⭐ Жақсы / 📚 Қанағаттанарлық / 💪 Жалғастыр

## v1.3 — Shuffle режимі (2026-02-23)

### ✨ Жаңа мүмкіндіктер

#### 🎲 Shuffle режимі

**🔑 Әкімші баптаулары:**
- 🎲 Сұрақтар араластыру (қосулы/өшірулі) — default: қосулы
- 🔀 Жауаптар араластыру (қосулы/өшірулі) — default: қосулы
- Әр қолданушы өз рет бойынша сұрақтарды көреді

**👤 Қолданушы баптаулары:**
- 🎲 Сұрақтар араластыру (қосулы/өшірулі)
- 🔀 Жауаптар араластыру (қосулы/өшірулі)
- Өз қалауы бойынша қосады/өшіреді

**Жұмыс принципі:**
- **Сұрақтар shuffle:** Тест бастағанда сұрақтар кездейсоқ ретпен беріледі
- **Жауаптар shuffle:** Әр сұраққа A,B,C,D опциялары кездейсоқ ретпен шығады
- **Дұрыс жауап автоматты сақталады:** Shuffle кезінде дұрыс жауап қайда болса да дұрыс есептеледі

## v1.2 — Quiz теңшеулер (2026-02-23)

### ✨ Жаңа мүмкіндіктер

#### 🔑 Әкімші (Бот иесі):

**⚙️ Баптаулар** (жаңа бөлім):

1. **🌐 Глобалды баптаулар**
   - 📝 Әдепкі сұрақтар саны: 5, 10, 15, 20, 25 (default: 15)
   - ⏱ Әдепкі уақыт лимиті: 10с, 30с, 60с, ∞ (default: 30с)
   - 🔢 Макс жауап опциялары: 4, 5, 6 (default: 4)
   - ✅ Макс дұрыс жауаптар: 1, 2, 3 (default: 1)

2. **📂 Санаттар басқару**
   - ➕ Санат қосу
   - ✏️ Санат өңдеу (атын өзгерту)
   - 🗑 Санат өшіру (растау + автоматты сұрақтарды өшіру)

3. **📋 Нұсқалар басқару**
   - ➕ Нұсқа қосу (санат → нөмір → атау → сипаттама)
   - 📋 Нұсқалар тізімі (санат бойынша)
   - 🗑 Нұсқа өшіру (сұрақтар сақталады, variant_id NULL болады)

#### 👤 Қолданушы:

**⚙️ Баптаулар** (жаңа батырма):
   - 📝 Сұрақтар саны: 5, 10, 15, 20, 25 (өз таңдауы немесе глобалды әдепкі)
   - ⏱ Уақыт лимиті: 10с, 30с, 60с, ∞ (өз таңдауы немесе глобалды әдепкі)
