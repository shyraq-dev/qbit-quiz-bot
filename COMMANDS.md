# QBit Quiz Bot — Барлық пәрмендер

## 👤 Қолданушы пәрмендері

### Telegram пәрмендері:
- `/start` — Ботты бастау, қош келдіңіз хабары
- `/help` — Бот туралы ақпарат
- `/stats` — Жеке статистика
- `/top` — Рейтинг кестесі (ТОП-10)

### Батырмалар (Main Menu):

#### 🎯 Тест бастау
- Категория таңдау терезесін ашады
- Категория таңдағаннан кейін 10 сұрақ басталады
- Streak жүйесі: 3 қатар дұрыс = +5 бонус ұпай
- Әр дұрыс жауап: +10 ұпай

#### 📊 Статистика
- Жалпы ұпай
- Ойындар саны
- Дұрыс/қате жауаптар
- Дәлдік пайызы (progress bar)
- Ең үздік streak
- Қазіргі streak
- Рейтингтегі орны (#N)

#### 🏆 Рейтинг
- ТОП-10 ойыншылар тізімі
- Әр ойыншы үшін:
  - Аты-жөні
  - ⭐ Жалпы ұпай
  - 🎯 Дәлдік %
  - 🔥 Үздік streak
- Өз орныңды көрсетеді (#N)

#### 💡 Сұрақ жіберу
**Қадамдық процесс:**
1. Категория таңдау (inline keyboard)
2. Сұрақты жазу (text)
3. A) жауап нұсқасы (text)
4. B) жауап нұсқасы (text)
5. C) жауап нұсқасы (text)
6. D) жауап нұсқасы (text)
7. Дұрыс жауапты белгілеу (A/B/C/D)
8. Алдын ала қарау
9. `/send` — жіберу немесе `/cancel` — болдырмау

**Статусы:** `pending` → Админ `approved`/`rejected` қылады

#### ℹ️ Туралы
- Бот туралы толық ақпарат
- Қалай жұмыс істейді
- Мүмкіндіктер тізімі

## 🛠 Админ пәрмендері

### Telegram пәрмендері:
- `/admin` — Админ панелін ашу

### Админ панелі батырмалары:

#### 📋 Кезектегі сұрақтар
- `pending` статусты сұрақтардың тізімі (5 дана)
- Әр сұрақта inline батырмалар:
  - **✅ Бекіту** → `approved` статусқа өзгертіп, `questions` кестесіне көшіреді
  - **❌ Қабылдамау** → `rejected` статусқа өзгертеді

#### ➕ Сұрақ қосу
**Қадамдық процесс (админдерге):**
1. Категория таңдау (inline keyboard)
2. Сұрақты жазу
3. A) жауап нұсқасы
4. B) жауап нұсқасы
5. C) жауап нұсқасы
6. D) жауап нұсқасы
7. Дұрыс жауап (A/B/C/D)
8. Қиындық деңгейі (inline keyboard):
   - 🟢 Жеңіл (easy)
   - 🟡 Орташа (medium)
   - 🔴 Қиын (hard)
9. Алдын ала қарау
10. `/save` — сақтау немесе `/cancel` — болдырмау

**Нәтиже:** Тікелей `questions` кестесіне `approved=1` күймен қосылады

#### 🗑 Сұрақ өшіру
**⚠️ ЖАҢА МҮМКІНДІК**

**Іздеу әдістері:**
1. **ID нөмірі бойынша:**
   - Мысалы: `123` жаз
   - Сұрақ тікелей табылады

2. **Кілт сөз бойынша:**
   - Мысалы: `Қазақстан` немесе `тарих`
   - Барлық сәйкес сұрақтар тізімі шығады (макс 10)
   - Біреу табылса — тікелей растау терезесі
   - Бірнешеу табылса — ID тізімі, нақты ID таңдау керек

**Процесс:**
1. Сұрақты іздеу (ID немесе кілт сөз)
2. Алдын ала қарау (толық сұрақ мазмұны)
3. Растау (inline buttons):
   - **✅ Иә, өшір** — сұрақты мәңгілік өшіреді
   - **❌ Жоқ, қалдыр** — болдырмау

**Ескертпе:** Өшірілген сұрақтарды қайтару мүмкін емес!

**Мысалдар:**
```
🔍 Сұрақ іздеу

ID бойынша: 47
Кілт сөз: Абай
Кілт сөз: технология Python
```

- Кезектегі сұрақтар саны

#### 🔙 Шығу

## 📝 Қосымша пәрмендер (FSM States ішінде)

### Сұрақ жіберу процесінде:
- `/send` — сұрақты жіберу (соңғы қадамда)
- `/cancel` — процесті тоқтату

### Админ сұрақ қосу процесінде:
- `/save` — сұрақты сақтау (соңғы қадамда)
- `/cancel` — процесті тоқтату (кез келген қадамда)

### Админ сұрақ өшіру процесінде:
- `/cancel` — өшіруді болдырмау (іздеу кезінде)

## 🎮 Ойын процесіндегі интерактивтер

### Quiz батырмалары:
- **A / B / C / D** — жауап таңдау (inline buttons)
- Бір сұраққа бір рет жауап беруге болады
- Жауаптан кейін 1.5 секунд автоматты келесі сұраққа өтеді

### Callback Data форматтары:
- `cat:{category}` — категория таңдау
- `ans:{question_index}:{letter}` — жауап беру
- `approve:{pending_id}` — сұрақты бекіту
- `reject:{pending_id}` — сұрақты қабылдамау
- `diff:{easy|medium|hard}` — қиындық таңдау
- `submit_cat:{category}` — жіберу үшін категория таңдау
- `delconfirm:{question_id}` — өшіруді растау
- `delcancel` — өшіруді болдырмау
